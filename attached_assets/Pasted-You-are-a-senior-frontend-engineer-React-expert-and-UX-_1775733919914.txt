## 🔴 1. CRITICAL BUG FIXES (HIGH PRIORITY)

A. Topic Edit / Menu Overflow Issue

Problem:

* The "⋯" (three-dot menu) for Topics is either:

  * Rendering outside viewport
  * Not visible
  * Not clickable
* Edit/Delete feels broken and inconsistent

Fix:

1. Ensure menu is always inside viewport:

   * Use position: absolute with boundary detection
   * If near right edge → open left
   * If near bottom → open upward

2. Add z-index:

   * z-index: 999 or higher
   * Ensure no parent has overflow: hidden blocking it

3. Fix clickability:

   * pointer-events: auto
   * No overlapping layers

4. Anchor positioning:

   * Use relative parent container
   * Menu should align to button properly

5. Add fallback:

   * If menu fails → show inline actions

---

B. Stale State in Focus Notifications

Problem:

* React async state causes wrong breakAction text in notifications

Fix:

const newAction = getBreakAction(duration);
setBreakAction(newAction);

notifyFocusEndBrowser(newAction?.text);

NEVER use stale state immediately after setState.
