# Project Memory

## Core
Dark theme study app. BG #0B0F14, card #1A222C, primary blue #3B82F6.
Inter font. Mobile-first max-w-768px. Bottom nav with 5 tabs.
Offline-first with localStorage. No backend.
No PWA service worker (causes issues in preview).
Brand: ExamFlowOS by imdvichrn. Subtle, premium, OS-feel.
